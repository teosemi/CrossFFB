Logitech G29 · CrossOver/Wine · macOS

Force Feedback for Windows racing games on macOS.

CrossFFB is a native macOS menu bar app that enables Logitech G29 Force Feedback for supported 64-bit Windows games running through CrossOver/Wine.

Built for a clean macOS experience.

Native bridge

A local macOS bridge talks directly to the Logitech G29 through HID.

Local proxy

The dinput8 proxy is installed next to the game executable, not inside Wine system folders.

No kernel driver

CrossFFB does not install a macOS driver, kernel extension, or background service.

Simple controls

Adjust Force Feedback gain and steering range directly from the menu bar.

Compatibility

CrossFFB is currently focused on Logitech G29 and 64-bit Windows games running through CrossOver/Wine.

  • Logitech G29
  • macOS 14 Sonoma or later
  • CrossOver or Wine
  • 64-bit Windows games

Tested games

CrossFFB has been tested mainly with Euro Truck Simulator 2 and Assetto Corsa Competizione.

  • Euro Truck Simulator 2
  • Assetto Corsa Competizione

Setup in three steps.

1

Install CrossFFB

Download the signed and notarized DMG from GitHub Releases.

2

Set dinput8 override

In the CrossOver/Wine bottle, set dinput8 to native,builtin.

3

Install the proxy

Use CrossFFB Setup and choose the folder containing the real game executable.

ACC note: choose the folder containing AC2-Win64-Shipping.exe, not only acc.exe.
